美饌 — 中式精緻料理

本館餐廳主打中式精緻料理，融合在地食材與經典烹飪技法，適合單人用餐、情侶晚餐與團體圓桌合菜。菜色包含海鮮、時令蔬菜、精緻小菜與湯品，並可依人數與需求客製化圓桌菜單。

圓桌合菜（範例）

  • 頭盤：冷盤拼盤
  • 熱菜：清蒸海魚、蠔油老菜心、三杯雞
  • 湯品：海味燉湯或中式燉品
  • 主食：白飯或炒飯
  • 甜點：時令水果或甜湯

如需預訂圓桌合菜或包廂服務，請於訂房或入住時事先告知餐廳。
