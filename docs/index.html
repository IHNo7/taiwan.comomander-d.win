陽明山上的溫泉旅宿體驗

國際大旅館歷史

國際大旅館創立於民國41年元月，係石造兩層日式風格之建築物半世紀以來我們提供著萬坪綠意與擁抱陽明山的美意！默默地服務著曾經駐足山上的朋友。在山頭裡迎賓入座足足有73年了，幾近人均壽命的長度，不管你是曾在斑駁的浴池中磨著背，曾被抓龍的手抓過兩回，或著你只是為了一圓兒時父親曾攜你來過的記憶也罷。國際大旅館永遠像個老朋友般，很自然的讓你毫不羞澀地接受著她，哪怕她已經蛻變、成長，賓至如歸的感覺依然熟悉不過‧‧‧

溫泉介紹

本館位於北投溫泉帶，擁有優質天然泉水，含多種礦物質，適合泡湯與放鬆。溫泉水經定期檢測與消毒， 旅客可於室內湯池與露天浴區享受溫泉療癒。
